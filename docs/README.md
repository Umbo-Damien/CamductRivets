# Documentation Images

This folder contains screenshots and images for the documentation.

## CAMduct Configuration Screenshots

### camduct-database-holes.png
Screenshot showing the CAMduct Database configuration for fixing holes:
- **Location**: Database > Fittings > Pattern Options > Holes tab
- **Key settings**:
  - Hole Diameter: 4.2 mm
  - Hole Spacing (Shoulder) Distance: 25.0 mm
  - Spacing: 110.0 mm
  - Draw Holes: ✓
  - Develop Holes: ✓

### camduct-takeoff-fixing-holes.png
Screenshot showing how to add "Fixing Holes" to the Takeoff Item Information:
- **Location**: Takeoff > Customize Main Takeoff > Item Information tab
- Shows "Fixing Holes" field in the available items list

## Adding Screenshots

To add the screenshots to this documentation:

1. Take screenshots of the CAMduct interface showing:
   - Database > Fittings > Pattern Options > Holes tab
   - Takeoff > Customize Main Takeoff > Item Information tab

2. Save them as:
   - `camduct-database-holes.png`
   - `camduct-takeoff-fixing-holes.png`

3. Place them in this `docs/images/` folder

## Image Requirements

- Format: PNG
- Resolution: Clear enough to read text
- Size: Reasonable file size (< 500KB per image)
